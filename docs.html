System Documentation

NexPY is a highly-sophisticated multi-bot orchestration system. It combines multiple state-of-the-art AI agents with a powerful plugin architecture to dominate the field.

The Plugin System

Our modular plugin architecture allows for nearly infinite extensibility. Plugins are written in Python and loaded dynamically at runtime.

  • Pre-Decision Hooks: Intercept game data before the bot makes a choice, allowing for custom situational overrides.
  • Post-Output Hooks: Modify the bot's controller outputs (steer, throttle, etc.) in real-time to add humanization or custom behaviors.
  • Situational Triggers: Built-in support for kickoff-specific logic, aerial detection, and supersonic handling.
